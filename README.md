# Manav Jan Kalyan Trust Website

Single-page Next.js 14 website for Manav Jan Kalyan Trust, built from the provided section references.

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```

## Stack

- Next.js 14 App Router
- Tailwind CSS v3
- Framer Motion
- Lucide React
- React Hook Form + Zod
- Vercel Analytics
